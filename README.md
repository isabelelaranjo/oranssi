## Funcionalidades Técnicas
- **Fuzzy Search (Busca por Aproximação):** Implementação do algoritmo de *Levenshtein* em JavaScript para sugerir correções de digitação (ex: encontrar "Ray-Ban" se o usuário digitar "Raiban").
- **Persistência de Dados de Lentes:** Seleção dinâmica de grau (OD/OE) e cor para lentes de contato, com captura de especificações para o checkout.
- **UX com Rolagem Controlada:** Lógica de navegação que prioriza a intenção do usuário, evitando rolagens automáticas indesejadas durante a digitação.
- **Cálculo Dinâmico de Frete:** Feedback em tempo real no carrinho sobre a meta de frete grátis (R$ 399,90).
- **Checkout via WhatsApp:** Integração com a API do WhatsApp para envio automático dos itens do carrinho e suas especificações.

## Tecnologias
- **HTML5 & CSS3:** Layout moderno, minimalista e com observações para Mobile First.
- **JavaScript Vanilla:** Lógica de negócio, manipulação de DOM e algoritmos de similaridade.
- **Deploy:** Vercel para hospedagem contínua.
